# 20K Design QA Scorecard

Use this before launch.

## Visual quality
- [ ] Every page has a point of view, not a template feel.
- [ ] Typography hierarchy is clear: display / section / card / body / label.
- [ ] Palette stays inside Teff Gold, Volcanic Rock, Ivory and restrained neutrals.
- [ ] No clutter, random colours or generic stock imagery.

## Motion and mobile
- [ ] Motion is subtle: 300–500ms, ease-in-out or luxury easing.
- [ ] No bouncing, playful animation or distracting transitions.
- [ ] Mobile is designed intentionally, not compressed from desktop.
- [ ] Tested at 375px, 767px, 1024px and desktop.

## SEO / AIO / GEO / LLM
- [ ] Each index and dynamic page has metadata.
- [ ] H1 is unique and descriptive.
- [ ] H2/H3 hierarchy supports skimmability.
- [ ] Internal links connect destinations, tours, guides, species and enquiry pages.
- [ ] JSON-LD exists where relevant.
- [ ] `/sitemap.xml`, `/robots.txt` and `/llms.txt` are live.

## Performance
- [ ] Hero video is compressed and has poster image.
- [ ] Real images are converted to AVIF/WebP.
- [ ] Lazy-load below-fold media.
- [ ] Avoid layout shift in card grids and hero sections.

## Conversion
- [ ] Every major page has a contextual enquiry CTA.
- [ ] WhatsApp CTA is visible but not aggressive.
- [ ] Trust signals are placed before high-commitment CTAs.

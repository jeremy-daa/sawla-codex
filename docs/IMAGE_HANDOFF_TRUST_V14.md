# Image Handoff — Trust Architecture V14

The code uses `PlaceholderImage` and descriptive labels. Replace placeholders only with licensed Sawla photography or approved commissioned assets.

| Route | Required hero | Supporting images |
|---|---|---|
| About | Real team/field image showing Ethiopia-based operations | founder-led leadership portrait (optional and approved), Addis office, team collaboration |
| Why Sawla | Guide/driver/traveler working together naturally | route briefing, field operations, specialist guiding |
| How We Work | Specialist reviewing itinerary/maps | proposal discussion, supplier confirmation, field briefing |
| Travel Specialists | Real journey design/operations team | planning call, route map, operations desk |
| Guides | Guide interpreting a site or habitat | cultural, birding, wildlife, trekking and photography specialists |
| Drivers & Vehicles | Driver with clean route-appropriate vehicle | 4×4 interior, minibus, luggage layout, daily vehicle check |
| Mobile Camps | Complete real camp in context | guest tent, sleeping arrangement, mess, kitchen, sanitation, power |
| Responsible Travel | Respectful interaction with visible consent/context | local guide, worship etiquette, wildlife viewing distance |
| Safety & Assurance | Guide/operations route briefing | first-aid kit, communications, vehicle check, pre-departure briefing |

## Minimum requirements

- hero crop: 1920×1080 or larger;
- Open Graph crop: 1200×630;
- WebP/AVIF derivatives;
- accurate descriptive alt text;
- photographer credit and rights record;
- model/property release where appropriate;
- no staged community portrait presented as spontaneous access;
- camp images labeled by configuration/date so one setup is not implied for every route.

# SEO / AIO / GEO / LLM Implementation — V4

## Strategic foundation

The website is structured around entity clarity rather than keyword stuffing. Core entities include:

- Sawla Tours
- Ethiopia tours
- Bespoke Ethiopia travel
- Lalibela
- Omo Valley
- Danakil Depression
- Simien Mountains
- Bale Mountains
- Ethiopian wolf
- Gelada
- Walia ibex
- Ethiopia travel guide
- Private Ethiopia itineraries

## Implemented architecture

- Destination hub and dynamic destination pages
- Tour style hub and itinerary pages
- Field guide hub and article pages
- Species library hub and wildlife pages
- Editorial Sawla Moments hub
- Testimonials hub
- Trip Finder
- Sawla Foundation / responsible travel positioning
- Sitemap and robots routes
- `llms.txt`
- JSON-LD schema support
- Canonical paths in metadata
- Internal linking through destinations, tours, species and guide content

## Metadata rules

For final content edits:

- Meta title: target 50–60 characters where possible
- Meta description: target 150–160 characters where possible
- One clear H1 per page
- H2s should structure meaning, not decoration
- Use descriptive internal links
- Avoid repeated generic phrases across page types

## Final SEO QA before launch

1. Crawl staging site with Screaming Frog or Sitebulb.
2. Check duplicate titles/descriptions.
3. Validate structured data with Schema.org / Rich Results tools.
4. Test sitemap and robots routes.
5. Confirm canonical URLs use the final production domain.
6. Add Google Search Console and Bing Webmaster Tools.
7. Verify image alt text after replacing placeholders.
8. Re-check Ethiopia logistics claims for current accuracy.
